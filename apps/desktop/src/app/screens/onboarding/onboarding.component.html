Let's get you set up

VideoDubber turns a video's spoken audio into another language — transcribe, translate, synthesize a new voice, and remux — all on this machine, fully offline once set up.

Before your first project we'll quickly:

  • Check this machine has everything it needs to run.
  • Let you pick your source & target languages and a transcription model.
  • Download the AI models for those languages. This needs an internet connection and can take several minutes depending on the model size.

The app, media tools and workers are already bundled — only the language models are fetched on first run.

This is a one-time setup.
